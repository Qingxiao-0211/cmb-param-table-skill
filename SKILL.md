# 招商银行参数表制作

根据基金公司标准参数表，自动填充招商银行公募基金产品上线导入模板。

## 工作流程

1. **获取输入文件**
   - 标准参数表（基金公司提供的源数据Excel）
   - 招行参数表模板（B列基本为空的导入模板Excel）

2. **运行转换脚本**
   ```bash
   python scripts/generate_param.py <标准参数表路径> <招行参数表模板路径> <输出路径>
   ```

3. **输出结果**
   - 生成填写完整的招行参数表Excel文件
   - B列已根据标准参数表数据自动填充
   - 保持原模板的格式和布局

## 核心功能

脚本自动完成以下映射：

### 产品基本信息
- 基金代码/主代码、产品简称、基金长名称 → 直接复制
- 封闭期参数/单位 → 从产品名称智能提取（如"6个月"→6, M:个月）
- 基金类型 → 根据产品类型自动映射（FOF→混合基金等）
- 基金状态 → 根据发行日期自动判断（发行期/交易期）
- 投资地区 → 根据QDII/FOF属性自动判断

### 费率信息
- 管理费率、托管费率、销售费率 → 直接复制
- 认购/申购费率 → 智能提取最高档（排除固定金额门槛）

### 账户信息
- 认购/申购清算账户 → 完整映射（账户名称、开户行、账号、支付行号）
- 回款账户（赎回款/分红款/认购退款）→ 自动关联对应账户

### 日期信息
- 募集起止日期 → 从发行日期范围自动解析
- 基金成立日期 → 募集期产品自动填1900-01-01

### 业务参数
- 确认日/交收日 → 从T+n/R+n格式自动提取数字
- 最低金额/份额 → 直接映射
- 分红方式/巨额赎回 → 布尔值转换

## 使用示例

**命令行方式：**
```bash
python scripts/generate_param.py \
  "标准参数表.xlsx" \
  "招行参数表模板.xlsx" \
  "招行参数表_已填写.xlsx"
```

**Python调用方式：**
```python
from scripts.generate_param import fill_zhaohang_template

fill_zhaohang_template(
    std_filepath="标准参数表.xlsx",
    template_filepath="招行参数表模板.xlsx",
    output_filepath="输出.xlsx"
)
```

## 注意事项

1. **标准参数表格式**：要求基金公司提供的标准参数表格式基本固定，包含字段名在第1列、值在第2列（账户信息类字段可能分布在第3-5列）
2. **招行模板格式**：要求使用招商银行提供的标准导入模板，A列参数名不可改动
3. **特殊产品调整**：脚本已处理大部分产品的自动映射，但对于特殊产品（如QDII、货币基金、定开产品等），生成后建议人工复核标黄和标蓝字段
4. **空格处理**：银行账号和支付行号会自动去除空格
5. **费率提取**：脚本会自动提取最高档费率，如果标准参数表的费率结构特殊，需人工复核
